Cafe Vibe 개인정보처리방침

카페의 소음 바이브를 공유하는 커뮤니티

1. 개요

Cafe Vibe(이하 "앱")은 카페의 소음 수준과 분위기를 기록·공유하는 iOS 앱입니다. 본 방침은 앱이 수집하는 정보, 사용 목적, 보관 기간 및 사용자 권리를 설명합니다.

시행일: 2026년 2월 27일

2. 수집하는 정보

  • 소음 수치(dB) — 마이크로 측정한 데시벨 숫자만 저장됩니다.
  • 위치 정보 — 앱 사용 중(When In Use)에만 현재 위치를 수집합니다.
  • 익명 사용자 ID — 로그인 없이 자동 발급되는 익명 식별자입니다. 개인 신원과 연결되지 않습니다.
  • 스티커 선택값 — STUDY / MEETING / RELAX 중 선택한 항목이 저장됩니다.
음성 녹음 파일은 절대 저장·전송되지 않습니다.
마이크는 데시벨 수치 추출 즉시 데이터를 휘발 처리합니다.

3. 수집하지 않는 정보

  • 음성 녹음 파일 또는 오디오 스트림
  • 연락처, 사진, 기타 민감 정보
  • 광고 식별자(IDFA) — 추적 허용 시에도 광고 목적으로 사용하지 않습니다.

4. 정보 이용 목적

  • 주변 스팟의 소음 평균 데이터 산출 및 지도 표시
  • 사용자별 리포팅 기록 및 통계 제공
  • 신뢰도 등급(Trust Score) 계산

5. 제3자 제공

수집된 정보는 원칙적으로 제3자에게 제공되지 않습니다. 단, 서비스 운영에 필요한 아래 제3자 서비스를 이용합니다.

  • Supabase — 데이터베이스 및 인증 (AWS ap-northeast-2 서울 리전)
  • Google Maps Platform — 지도 표시
  • Google Maps Platform — 지도 표시 (위치 정보 전송)

6. 보관 기간

  • 소음 리포트 데이터: 마지막 활동일로부터 30일 이후 지도에서 비표시
  • 회원 탈퇴 시: 모든 리포트 데이터 즉시 삭제
  • 위치 정보: 앱 세션 종료 시 미저장

7. 사용자 권리

  • 데이터 열람 — 앱 내 마이페이지에서 본인의 측정 기록 확인 가능
  • 데이터 삭제 — 앱 설정 → "계정 삭제"로 모든 데이터 즉시 삭제
  • 위치 권한 철회 — 기기 설정에서 위치 권한 언제든지 거부 가능
  • 마이크 권한 철회 — 기기 설정에서 마이크 권한 언제든지 거부 가능

8. 어린이 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않습니다. 만 14세 미만은 보호자의 동의 없이 앱을 사용하지 않도록 안내합니다.

9. 방침 변경

본 방침이 변경될 경우 앱 업데이트 또는 공지사항을 통해 사전 안내합니다. 변경 방침은 공지일로부터 7일 후 적용됩니다.
